Proyectos Personales

MeetingBurner | Real-Time Meeting Cost Visualizer

Aplicación web diseñada para cuantificar y visualizar el impacto financiero de las reuniones. El sistema guía al usuario a través de un flujo de tres etapas ("Setup - Track - Report") para transformar el tiempo en métricas de negocio tangibles.

Funcionalidades principales:

  • Configuración dinámica: Interfaz intuitiva para parametrizar asistentes, tarifas/hora y divisa.
  • Dashboard en tiempo real: Visualización de datos en vivo mediante gráficos acumulativos y contadores de alto rendimiento.
  • Generación de reportes: Emisión automática de un "ticket de reunión" con el resumen de costes y estadísticas finales.

Aspectos técnicos:

  • Stack: React + Vite para un rendimiento instantáneo.
  • UI/UX: Diseño moderno "Dark Mode" utilizando Tailwind CSS y componentes shadcn/ui.
  • Data Viz: Implementación de librerías gráficas (Recharts) y animaciones fluidas (Framer Motion).

Aura Private Flights | High-Fidelity Luxury Travel Interface

Landing page de alta fidelidad diseñada para un servicio exclusivo de chárter de jets privados. El proyecto captura la esencia del "Lujo Silencioso" mediante una estética minimalista y una experiencia de usuario inmersiva, enfocada en transmitir prestigio y confort a clientes de alto perfil.

Funcionalidades principales:

  • Experiencia Inmersiva: Diseño visual de alto impacto con una sección "Hero" envolvente y animaciones activadas por scroll.
  • Showcase de Flota: Visualización interactiva y elegante de las aeronaves disponibles mediante sliders dinámicos.
  • Niveles de Membresía: Presentación estructurada de planes y precios con un diseño claro y sofisticado.

Aspectos técnicos:

  • Stack: React + Vite para un rendimiento y navegación optimizados.
  • UI/Styling: Diseño responsivo y moderno implementado con Tailwind CSS.
  • Motion: Animaciones fluidas y transiciones de elementos orquestadas con Framer Motion.
